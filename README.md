# 🏠 Analyseur d'Investissement Immobilier — Edmonton

Application web d'analyse automatique de pro forma immobiliers pour évaluer des opportunités d'investissement à Edmonton, Alberta.

## Fonctionnalités

- 📊 Import Excel (.xlsx) et PDF
- 🏘️ Analyse de quartier (vacance, sécurité, accessibilité)
- 💰 Comparaison revenus projetés vs marché
- ⚠️ Vérification des dépenses (minimums recommandés)
- 📋 Pro forma révisé automatique
- 🎯 Score d'investissement sur 100
